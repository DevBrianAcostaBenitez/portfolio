Banki

Una aplicacion que simula operaciones bancarias, no se usa dinero real. podemos hacer ingresos, retiros, transferencias, solicitar tarjetas de credito y cancelarlas, establecimiento. Se realizo con kotlin y se usa la base de datos no relacional Firebase
